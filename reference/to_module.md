# Parse 'RAVE' Module and Returns Parsed Content in Environments

Parse 'RAVE' Module and Returns Parsed Content in Environments

## Usage

``` r
to_module(
  module_id,
  sidebar_width = 3,
  parse_context = c("rave_running_local", "rave_running")
)
```

## Arguments

- module_id:

  module ID

- sidebar_width:

  input sidebar width

- parse_context:

  parse context, default is `"rave_running_local"`
