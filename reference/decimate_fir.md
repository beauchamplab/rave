# Decimate or Down-sample a Signal using `FIR` Filters

Decimate or Down-sample a Signal using `FIR` Filters

## Usage

``` r
decimate_fir(x, q, n = 30)
```

## Arguments

- x:

  signal to be decimated

- q:

  integer factor to decimated by

- n:

  filter order used for down-sample procedure, default is 30

## Value

Down-sampled signal
