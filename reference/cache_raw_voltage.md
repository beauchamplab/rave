# Import Subject "Matlab" File and Create "HDF5" files

Import Subject "Matlab" File and Create "HDF5" files

## Usage

``` r
cache_raw_voltage(project_name, subject_code, blocks, electrodes, ...)
```

## Arguments

- project_name:

  project name

- subject_code:

  subject code

- blocks:

  blocks to be imported

- electrodes:

  to be imported (please check file existence before calling this
  function)

- ...:

  ignored
