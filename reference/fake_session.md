# Fake 'shiny' Session for Debug Purpose

Fake 'shiny' Session for Debug Purpose

## Usage

``` r
fake_session(rave_id = "__fake_session__", id = NULL)
```

## Arguments

- rave_id:

  internally used

- id:

  module ID, used to create scope, will passed to
  [`NS`](https://rdrr.io/pkg/shiny/man/NS.html)

## Value

Fake shiny session for debug purpose
