# Create S3 Generics that Respects 'RAVE' Context

Create S3 Generics that Respects 'RAVE' Context

## Usage

``` r
rave_context_generics(
  fun_name,
  fun = function() {
 }
)
```

## Arguments

- fun_name:

  generic function name

- fun:

  function that set the arguments of the generic

## Value

A generic function
