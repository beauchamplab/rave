# Returns wavelets to be used for wavelet function

Returns wavelets to be used for wavelet function

## Usage

``` r
wavelet_kernels(freqs, srate, wave_num)
```

## Arguments

- freqs:

  vector of center frequencies for decomposition

- srate:

  sample rate (in Hz)

- wave_num:

  desired number of cycles in wavelet (typically 3-20 for frequencies
  2-200).
